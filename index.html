Getting started

HTML

The chat heads will be constrained to a container element in your HTML.

<div class="chat-head-container">
</div>

Include the Chat Heads JavaScript and CSS in your site.

<script src="/path/to/chat-heads.min.js"></script>
<link rel="stylesheet" href="path/to/chat-heads.css">

Initialize with JavaScript

Initialize the Chat Heads plugin, specifying any required options.

var chat = new ChatHeads({
    requireEmail: true
});

Now it's time to start adding heads!

Methods

addHead

Each chatting user 

chat.addHead(new ChatHead({
    name:  'Colin',
    email: 'colin@squarecat.io'
}));
